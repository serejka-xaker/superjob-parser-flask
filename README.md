# Проект: Парсер вакансий SuperJob с веб-интерфейсом

**Сентябрь 2024 | Фриланс | Full-cycle разработка**

- Самостоятельно провёл сбор требований, спроектировал архитектуру и реализовал проект от анализа задачи до передачи клиенту.
- Разработал локальное веб-приложение на Flask с HTML/CSS-интерфейсом для управления процессом парсинга через браузер.
- Реализовал полный OAuth2-процесс авторизации через SuperJob API с получением и обновлением токенов и контролем срока их действия.
- Спроектировал хранение токенов в SQLite с обновлением и проверкой валидности.
- Реализовал отказоустойчивую HTTP-сессию для обработки ошибок.
- Внедрил многопоточность для запуска парсинга в фоновом режиме без блокировки веб-интерфейса.
- Реализовал фильтрацию и структурирование данных вакансий.
- Настроил экспорт результатов в Excel.
- Скомпилировал приложение в EXE-файл для передачи заказчику.

### Стек
`Python`, `Flask`, `requests`, `SQLite`, `threading`, `BeautifulSoup`, `pandas`, `OAuth2`, `PyInstaller`
